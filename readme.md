# 背景介绍
worktime-tools，是为了解决工时系统如下不足而开发：
* 员工无法及时准确的统计自己的工时使用情况。
* 容易出现到月底才发现工时不足导致扣工资。
* 拒绝无谓的加班和浪费时间和生命，

# 功能
* 可以随时随地的统计截止当前的剩余可抵扣时长和需增补时长
* 支持手工录入当日上下班时间做到准实时统计
* 支持剩余抵扣工时展示，以及告警提示
* 当前只支持.xlsx文件格式的导入
* 当前只支持未出现过早退和迟到情况的统计，针对迟到和早退情况自动过滤，后续添加此场景的显示
* 当前只支持通过命令行方式导入和统计
* 后续将提供web版本,APP版本的导入，统计，录入功能
